# 💖 Valentine Mini Website

A cute and playful mini React website that asks the most important question:

**“Will you be my Valentine?”**

Built with React + Vite, featuring fun button interactions, animations, and a mobile-friendly layout.

---

## ✨ Features

- GIFs and romantic vibe
- **No** button disappears when clicked
- **Yes** button smoothly scales up
- Final celebration screen after saying Yes
- Hover & click animations on buttons
- Fully responsive & mobile-friendly

---

## 🛠 Tech Stack

- **React**
- **Vite**
- **JavaScript**
- **CSS (Flexbox + animations)**
